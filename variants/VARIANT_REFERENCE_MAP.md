# AUN Variant Reference Map

15案を個別構造へ作り直すための参照メモです。`awesome-design-md-global` の参照先、`impeccable-style` で強める観点、`kamui t2i` で足す素材の役割を先に固定します。

## Usage

- `Awesome Design MD GLOBAL` は構造と空気の起点だけに使う
- `Impeccable Style` は `layout / colorize / typeset / adapt / polish` を中心に使う
- `kamui t2i` は差を装飾で埋めるのではなく、各 layout の読み順を補強する役割で使う
- 画像生成前に、`hero / break / detail / studio` のどれを作るかを決める

## Variant Map

- DESIGN refs: `jp/droga5`, `global/ferrari`, `global/lamborghini`

  - hero: hard-shadow studio portrait with black void and blood-red cue
